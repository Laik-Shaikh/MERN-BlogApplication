<h4>If you want to use this project in your machine</h4>
<br>
<h3>Installations</h3>
<ul>
<li>Install Node Js</li>
<li>Install Git</li>
<li>Install Vs Code or any other Code Editor</li>
</ul>

<h5>Goto Mongodb cloud atlas and create your account</h5>

<h5></h5>
<h5>Go inside any directory of your machine</h5>
<h5>Right Click and select git bash here</h5>
<h5>now do git clone</h5>
<h5>go inside the cloned folder</h5>
<h5>Go inside server folder</h5>
<h5>create a .env file</h5>
<h5>and write this inside .env file </h5>
<pre>
DB_USERNAME = enter your user name from mongodb cloud
DB_PASSWORD = enter your mongo db password
DB_DATABASE_NAME = enter your database base name 
DB_CLUSTER_NAME = enter your cluster name
ACCESS_SECRET_KEY = your access secret key
REFRESH_SECRET_KEY = your refresh secret key
</pre>

<h5>Go inside Server folder</h5>
<h5>Open terminal in that folder</h5>
<h5>and type npm install</h5>
<h5>then type npm start</h5>

<h5>Go inisde Client Folder</h5>
<h5>Open terminal in that folder</h5>
<h5>type npm install</h5>
<h5>then type npm start</h5>

<h5>This should start the application in your browser</h5>
